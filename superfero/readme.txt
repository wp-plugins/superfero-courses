Superfero Widget grabs the latest online courses from superfero.com to display on your sidebar

== Description ==

Superfero.com is an online implementation platform that will multiply the impact of everyone working with groups of people. Superfero works by combining social group features with a narrow focus on solving challenges within deadlines.

This widget grabs the latest online courses from superfero.com to display on your sidebar.

== Installation ==

Installing "Superfero" can be done either by searching for "Superfero" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
